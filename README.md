Tool to draw on an 8k jpg file of the GTAV map.
2 versions available: Satellite and Roads, couldnt bother to make a system to bundle them into one.
Fully written in Python. 


HOW TO USE:
1. Download Source Code from here.
2. Look inside GTAMapEditorBuilds/v1.1
3. 2 .exes available, Satellite and Roadmap
4. Open the .exe

CONTROLS:
- Hold Middle Mouse Button to pan the map
- Hold Left Click to draw
- Hold ctrl + Left Click to erase
- Mouse wheel to zoom
